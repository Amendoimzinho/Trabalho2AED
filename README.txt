Quando iniciao o Programa ir em '2 - Outros' -> '1 - Criar arquivo de dados' para criar o arquivo binario

======= Menu Principal =======
1 - Menu de Livros            // Abre o menu de operacoes para livros 
2 - Outros                    // Abre o menu de operacoes especiais

==============================


======= Menu Livros =======
1 - Remover livro           // Pede um ID para remover
2 - Listar livros           // Lista todos os livros inOrder
3 - Listar registros livres // Imprime a lista livre
4 - Total de livros         // Imprime o total de livros cadastrados
0 - Voltar                  // Volta pro menu Principal
===========================



======= Menu Outros =======
1 - Criar arquivo de dados  // Volta pro menu inicial
2 - Ler arquivo de texto    // Pede e le um arquivo
3 - Imprimir niveis         // Imprime a arvore de dados por nivel
0 - Voltar                  // Volta pro menu Inicial
